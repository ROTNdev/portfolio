ReefMC (Website)

Early March 2014, I was invited by a good friend of mine to work with ReefMC to help develop its multiserver network. My challenges were to help interconnect the network by building a core set of plugins with custom systems such as ban management solutions and network-wide player to player interactions (e.g. cross server private messaging and staff chat). Since these changes, the player response and active player base is better than it ever has been and continues to grow.

ImpulseNetwork

December 2013, I took upon the challenge of building a Network, from scratch. I was brought in as Developer, by two YouTuber’s (PixelSwift/SubZeroExtabyte - Two of the three founders). My goal/aim was to build a network, on a dedicated machine, that had Bungee integration and 4 unique game modes. This goal was achieved with a great level success.

MCTheShip

April 2013, when I wanted to develop a Minigame server, like Trouble in Mineville. I learnt of the game “The Ship” and wanted to recreate it in Minecraft. At this time, I was very ambitious and really didn’t know a lot about java.

MCTheShip was open for 4 months, and did attract some attention. From the players that did play on the server, they really enjoyed it. I managed to create a complete working replica, however it is a shame I had to close down the server.

PrisonMC

PrisonMC was a few months after MCTheShip. A friend of mine wanted to create a prison server, and he wanted my help. Simply, the task was to create a custom prison plugin, which controls everything (Mine Resets, Guard Features, and Ranking systems etc.). My friend didn’t want to use generic plugins, rather one plugin from me that game the same feel throughout the entire server. The server shut down a few months later due to lack of funds; however my Plugin was completed and fulfilled its duties.

CraftGasm (Website)

DragonEntrance

When a player joins the server for the first time, they get flown in to the spawn point by riding an EnderDragon. This looks quite spectacular and has never been done before. This plugin uses MySQL for support of multiple hubs (so you only can ride the dragon once), as well as custom mob creation/management.

Warps

Warps was a MySQL focused plugin which does as it's name suggests. It allowed the player to create up to 9 warps, and less/more depending on their permissions. It had advanced full inventory controls/interactions which made it easy for players to use. The warp data was loaded/saved onto MySQL for ease of storage.

CrystalFortune

CraftGasm has a virtual economy called "Crystals". CrystalFortune allows the player to 'roll the dice' once every 24 hours, in the hope to win a bonus of Crystals every day. The player 'rolled the dice' by using an Inventory solution.

InventoryStats

CraftGasm has a fully functioning statistics API and system in place. InventoryStats uses that existing framework to get the player's statistics and display it in an inventory.

SkyServers (Website)

FreezeTag

FreezeTag is a simple replica of the childhood game of "Freeze" or "Stuck in the Mud". Essentially, the tagger has to tag all players to win. When a player is tagged, they cannot move. When they are not tagged, they can un-tag tagged players.

StrayyaPrison
